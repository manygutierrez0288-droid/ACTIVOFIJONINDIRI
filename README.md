# SIAFNIN - Sistema de Inventario de Activos Fijos

## Descripción
SIAFNIN es un sistema integral para la gestión y control de inventarios, específicamente diseñado para el manejo de activos fijos, vehículos y terrenos.

## Módulos Principales
- **Activos Fijos**: Gestión completa del ciclo de vida de los activos.
- **Vehículos**: Control de flota, asignaciones y depreciación.
- **Terrenos**: Registro y administración de propiedades.
- **Reportes**: Generación de reportes detallados y auditoría.

## Tecnologías
- **Backend**: Laravel
- **Frontend**: Vue.js (Inertia.js)
- **Base de Datos**: MySQL

## Instalación
1. Clonar el repositorio.
2. Ejecutar `composer install`.
3. Ejecutar `npm install` y `npm run dev`.
4. Configurar el archivo `.env`.
5. Ejecutar migraciones: `php artisan migrate`.
